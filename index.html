Welcome to MyPortal!

The testing branch isn't intended for use, with a few exceptions:

  • You are an administrator, and are editing features that are experimental, and shouldn't be in the mainstream release yet
  • You want to see potential new features before they come to the mainstream

It is possible that some features in the testing branch will be removed in the future, and never be worked on again.

It is also possible that in the near future that the features will come to the main branch.


It isn't recommended to add this branch to an auto-update script. It may break features that you are working on yourself, if you are using this test branch.
